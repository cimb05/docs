# vi 编辑器


- 复制命令

| 快捷键| 功能描述|
| - | - |
| y| 复制已选中的文本到剪贴板|
| n+yy| 复制光标所在行，此命令前可以加数字 n，可复制多行|
| yw| 复制光标位置的单词|
| ctrl+v+方向键+yy| 先按ctrl+v，然后按方向键选中区块，最后按下yy复制|


- 查看指定内容

| 快捷键| 功能描述|
| - | - |
| /| 开始进行搜索|
| n| 查找下一个|


- `dd`
  删除当前行

- `i `
  进入编辑模式

- `ESC`
  退出编辑模式

- `:wq`
  保存

- `q!`
  强制退出

- `ctrl+s`
  锁屏模式

- `ctrl+q`
  退出锁屏
